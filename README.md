# JHIPSTER REGISTRY

JHIPSTER REGISTRY Service Deployment Guide

## Installation

If you have already installed the project

```bash
$ cd jhipster-registry/
$ ./mvnw
```

If this is your first installation

```bash
$ cd jhipster-registry/
$ npm install
$ ./mvnw
```

## Launch the project

```bash
$ cd jhipster-registry/
$ ./mvnw
```

to launch the user interface <http://127.0.0.1:8761>

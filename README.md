# JHIPSTER REGISTRY

Guide de deploiement du service JHIPSTER REGISTRY

## Installation

Si vous avez deja installé le projet

```bash
$ cd jhipster-registry/
$ ./mvnw
```

Si c'est votre première installation

```bash
$ cd jhipster-registry/
$ npm install
$ ./mvnw
```

## Lancer le projet

```bash
$ cd jhipster-registry/
$ ./mvnw
```

pour lancer l'interface utilisateur <http://127.0.0.1:8761>
